Crear una Base de datos

Hay muchas formas de crear una BD, haciendolo manualmente, usando un script, clonando una base de datos desde un repositorio, etcétera, de momento la crearemos manualmente o ejecutaremos el script. puedes bajar uno de ejemplo de esta base de datos creada haciendo clic AQUI

Creación manual de tablas

Paso 1. Con nuestro servicios de APACHE y MySQL iniciados ingresamos a phpMyAdmin y hacemos clic en New del panel de operaciones a la izquierda


Paso 2. Ingresamos el nombre de nuestra BD, el cual será crud_php y hacemos clic en create


Paso 3. Al crear correctamente nuestra BD, creamos una tabla llamada usuarios y agregamos los siguientes campos con sus respectivos tipos y tamaños: id tipo INT, nombre, apellido e email de tipo VARCHAR (50)


Adicional a esto para la columna id marcamos la opción para que no permita valores nulos y sea autoincrementable


Paso 4. Hacemos clic en save para guardar la tabla

Ejecutar el script

Al tener un script todo es mucho mas sencillo, para crear la base de datos solo debes ejecutar el script que descargaste de la siguiente manera.

Paso 1. Con nuestro servicios de APACHE y MySQL iniciados ingresamos a phpMyAdmin y hacemos clic en la pestaña de SQL


Paso 2. En el cuadro de texto pegamos el contenido de nuestro script


Paso 3. Hacemos clic en GO para ejecutar el script


El script generara la base de datos con la estructura correcta para realizar nuestro CRUD.
