Formulario HTML y CRUD

En esta practica construiremos un formulario HTML simple para realizar nuestro CRUD, cuando usamos la palabra CRUD, nos referimos al manejo que podemos darle a los datos, Create (crear), Read (leer), Update (actualizar), Delete (eliminar)

Creación manual de tablas

Paso 1. Lo primero que debemos hacer es crear nuestra carpeta en el directorio de publicacion local de APACHE, el cual por defecto está en la siguiente ubicacion: C:\xampp\htdocs, la carpeta tendra por nombre crud, dentro de esta creamos una carpeta llamada php y una llamada procesos.


Paso 2. Descargamos un editor de textos como Notpad++ o Sublime text y creamos el archivo de conexión con la base de datos.


la variable $mysqli almacena la nueva conexión con la base de datos crud_php, el primer, segundo y tercer parametros ("localhost","root" y "") son la configuración por defecto para el host, el username y el password de la base de datos. mas info AQUI en caso de error en al conexión se mostrara el mensaje "Este sitio esta presentando problemas". guardamos el archivo con el nombre connect.php dentro de la carpeta php


Paso 3. Creamos el archivo de presentación el cual tendra nuestras opciones principales crear, buscar y listar.


en este archivo creamos los enlaces para nuestras opciones, las etiquetas href creadas son los proximos archivos que debemos crear, guarda este archivo como index.php, dentro de la carpeta crud


Al ejecutar la URL del index http://localhost/crud/index.php debemos ver los siguiente:


Paso 4. Creamos las vistas y los controladores de los enlaces del punto anterior.


4.1 guardar.php


Este archivo contiene el formulario html para ingresar el nombre, apellido e email, los cuales hacemos obligatorios con la etiqueta required="required" y en el caso de email validamos que sea el texto tenga una estructura de correo electronico con la etiqueta type="email", estos son los datos que almacenaremos en la BD, el botón submit envia estos datos al controlador guardarDatos.php donde se realizara la inserción a la base de datos. guardamos este archivo en la carpeta crud


en nuestro navegador debemos ver el formulario de la siguiente manera


4.1.1 guardarDatos.php


Este archivo contiene la referencia para la conexión a la base de datos y la capura de los datos enviados por el formulario, los cuales llegan en la variable $_POST, despues mediante el comando "query" de mysqli ejecutamos la consulta para guardar los datos en la BD. guardamos este archivo en la carpeta procesos

En esta vista se listaran todos los registros de la tabla usuarios de nuestra base de datos. en la linea 22 se observa la consulta que trae todos los datos de la tabla y como usando un ciclo while construimos cada fila de la tabla, ademas agregamos los enlaces para actualizar o eliminar cada registro


Este archivo lo debemos guardar en la carpeta crud
